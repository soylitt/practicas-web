¿Que debe saber un diseñador gráfico si desea ser diseñador web?

(Diseñador Full Stack)

Con el paso del tiempo, ser diseñador de páginas web ha pasado de estar centrado en el diseño gráfico a estar más enfocado en la programación, o al menos en el diseño de lenguajes de descripción de páginas o HTML y formatos visuales que es el CSS.

Para un diseñador gráfico que desea convertirse en diseñador web, es importante adquirir una serie de habilidades y conocimientos adicionales. A continuación, se detallan los aspectos clave que un diseñador gráfico debe saber para realizar una transición exitosa al diseño web:


Qué es un diseñador Full Stack

El desarrollador Full Stack se encarga de crear y mantener las partes de front y back end de una aplicación web, con polivalencia para el conocimiento de lenguajes de programación y sistemas operativos. Su objetivo es coordinar los procesos de desarrollo y producción de software.

Conocimientos de un diseñador web

  • Dominio de softwares de diseño gráfico
  • Manejo de teoria del color, tipografías, clasificación de signos, temas de composición
  • Conceptos básicos
  • Lenguajes HTML, CSS y JS.
  • Principios de usabilidad y diseño centrado en el usuario
  • Frameworks de extensión de Javascript como jQuery, etc.
  • Conocimientos de SEO
  • Conceptos básicos de publicidad y diseño corporativo
  • Shared Hosting, FTP, cPanel y configuración básica de un sitio en la nube
