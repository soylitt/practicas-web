¿Qué es internet?

Internet es una red global de computadoras interconectadas que permite la comunicación y el intercambio de información entre usuarios de todo el mundo. Se trata de una red descentralizada que conecta millones de dispositivos y sistemas informáticos en todo el planeta.



¿Qué son los Proveedores de Internet?

Los proveedores de Internet (ISP, por sus siglas en inglés de Internet Service Provider) son empresas u organizaciones que ofrecen servicios de acceso a Internet a individuos, hogares, empresas u otras organizaciones. Estos proveedores mantienen la infraestructura necesaria para conectar a los usuarios a Internet, lo que incluye equipos de red, servidores y conexiones de alta velocidad a la red global de Internet.



Tipos de proveedores de internet.

Proveedores de servicios de Internet por cable (Cable ISP) Proveedores de servicios de Internet por fibra óptica (Fiber ISP)
Proveedores de servicios de Internet por DSL (Digital Subscriber Line ISP) Proveedores de servicios de Internet por satélite (Satellite ISP)
Proveedores de servicios de Internet móvil (Mobile ISP) Proveedores de servicios de Internet comunitarios o locales


Protocolos de Aplicación

Los protocolos de aplicación son un conjunto de reglas y convenciones que rigen la comunicación y el intercambio de datos entre aplicaciones de software en una red de computadoras. Estos protocolos definen el formato de los mensajes intercambiados, así como los procedimientos para el inicio, mantenimiento y finalización de las comunicaciones.



Carpeta local VS Carpeta remota

Carpeta Local

  • Son las carpetas que residen en el dispositivo de almacenamiento de una computadora o dispositivo.
  • Están físicamente almacenadas en el disco duro u otro tipo de almacenamiento local, como un SSD o un dispositivo de almacenamiento externo conectado directamente al equipo.
  • Los archivos en estas carpetas se pueden acceder directamente desde el dispositivo donde están almacenados, sin necesidad de una conexión a Internet o a otra red.

Carpeta Remota

  • Son carpetas que residen en un servidor o dispositivo de almacenamiento remoto, generalmente ubicado en otro lugar físico y accesible a través de una red, como Internet.
  • Los archivos en estas carpetas se almacenan en un servidor remoto y se accede a ellos mediante protocolos de red, como FTP, SFTP, SMB (Windows File Sharing), NFS (Network File System), entre otros.
  • Para acceder a los archivos en carpetas remotas, es necesario tener una conexión de red activa, como una conexión a Internet.


¿Qué es interfaz de usuario?

Una interfaz de usuario, comúnmente abreviada como UI (del inglés User Interface), es el medio a través del cual un usuario interactúa con un dispositivo, aplicación o programa de computadora. Esta interacción puede incluir elementos visuales, como botones, menús, ventanas y gráficos, así como también elementos táctiles, gestuales o de voz en interfaces más avanzadas. El objetivo de una interfaz de usuario es facilitar la comunicación entre el usuario y el sistema, permitiendo que el usuario pueda controlar y utilizar el dispositivo o programa de manera eficiente y efectiva.



¿Qué son los protocolos de internet?

El Protocolo de Internet (IP) es un protocolo, o conjunto de reglas, para enrutar y direccionar paquetes de datos para que puedan viajar a través de las redes y llegar al destino correcto. Los datos que atraviesan Internet se dividen en trozos más pequeños, llamados paquetes.

La información IP se adjunta a cada paquete y esta información ayuda a los enrutadores a enviar los paquetes al lugar correcto. A cada dispositivo o dominio que se conecta a Internet se le asigna una dirección IP y a medida que los paquetes se dirigen a la dirección IP adjunta, los datos llegan a donde se necesitan.



¿Qué es Html?

HTML, siglas en inglés de Hypertext Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje estándar utilizado para crear y diseñar páginas web. Fue creado por Tim Berners-Lee en la década de 1990 como parte del proyecto World Wide Web (WWW) en el CERN. HTML proporciona una estructura básica para el contenido de una página web, utilizando etiquetas o elementos que indican cómo se debe mostrar ese contenido en un navegador web. Estas etiquetas pueden utilizarse para definir títulos, párrafos, listas, enlaces, imágenes, formularios y muchos otros elementos que componen una página web.



¿Qué es la semántica en Html?

La semántica en HTML se refiere al significado o la estructura lógica que se le da al contenido de una página web mediante el uso adecuado de elementos HTML. En otras palabras, se trata de utilizar los elementos HTML de manera que reflejen el propósito y la función del contenido que están marcando. Por ejemplo, en lugar de simplemente usar un elemento div> genérico para agrupar contenido, es preferible utilizar elementos semánticos como header>, nav>, main>, section>, article>, footer>, etc., según corresponda. Esto ayuda a los motores de búsqueda y a otros programas a comprender mejor la estructura y el significado de la página, lo que puede tener beneficios en términos de accesibilidad, SEO (Optimización para Motores de Búsqueda) y mantenimiento del código.
